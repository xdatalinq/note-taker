  ## Description
  Note Taker that can be used to write and save notes. This application uses an Express.js back end and will save and retrieve note data from a JSON file. 
  Deployed via Heroku, tests included (Jest).

    Built Using:
    
    JavaScript
    Node.js
    Express.js
    Jest
